# 🌐 Simple 3-Page Bootstrap Website

This is a beginner-friendly web development project built using **HTML** and **Bootstrap**. It includes three main pages: a Home page, a Registration page, and a Login page with responsive design.

---

## 📌 Features

* 🏠 **Home Page**

  * Responsive layout using Bootstrap
  * Navigation bar
  * Welcome section

* 📝 **Registration Page**

  * User-friendly form using Bootstrap components
  * Input fields: Name, Email, Password, etc.
  * Responsive design

* 🔐 **Login Page**

  * Clean and simple login form
  * Email and password input fields
  * Styled with Bootstrap

---

## 🛠️ Technologies Used

* HTML5
* Bootstrap 5
* CSS3 (optional customization)
* JavaScript (optional validation)

---

## 🚀 How to Run

1. Clone the repository:

   ```
   git clone https://github.com/Shikha/bootstrap.git
   ```

2. Open the project folder.

3. Open `index.html` in any web browser.
